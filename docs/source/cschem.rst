.. _cschem:

User guide: Designing electronic circuits with CSchem
=====================================================

This document describes the usage of “CSchem” a (hopefully)
easy-to-use program for drawing electronic circuit diagrams. A
companion program, “CPCB,” is provided for instantiating circuits on a
PCB. This program is documented in a :ref:`separate user guide <cpcb>`.
