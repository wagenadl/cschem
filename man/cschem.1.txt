CSCHEM(1)
=========
:doctype: manpage

NAME
----
cschem - Electronic Circuit Design

SYNOPSIS
--------
*cschem* ['SCHEMFILE']

DESCRIPTION
-----------
The *cschem*(1) command starts the CSchem electronic circuit design
editor. Optional argument specifies a circuit to load.


OPTIONS
-------
None.

BUGS
----
CSchem is in active development. There may be bugs. Please
contact the author if one bites you.
