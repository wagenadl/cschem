CPCB(1)
=======
:doctype: manpage

NAME
----
cpcb - PCB Layout

SYNOPSIS
--------
*cpcb* ['CPCBFILE']

DESCRIPTION
-----------
The *cpcb*(1) command starts the CPCB PCB layout
editor. Optional argument specifies a layout to load.


OPTIONS
-------
None.

BUGS
----
CPCB is in active development. There may be bugs. Please
contact the author if one bites you.
